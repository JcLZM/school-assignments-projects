Instructions:

This is a java program, so please have or install JDK.
1. Open command prompt (Windows key + R, type cmd)
2. Navigate to folder containing .java file
3. type "javac " followed by the file name (i.e. "javac greedyAlgo.java")
4. type "java" followed by the file name, but without .java, followed by the name of the data text file (i.e. "java greedyAlgo a3data.txt")
5. Results should be outputted.
